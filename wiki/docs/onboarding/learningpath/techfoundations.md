Understand the core technologies used in Structra. 

- **Next.js:** A React framework for building server-side rendered and static web applications. Compare it to vanilla React to understand its benefits in SEO and performance. 
- **React Native & Expo:** A framework for building native mobile apps with React. Expo simplifies the development process. Compare it to native Android (Java/Kotlin) or iOS (Swift) development. 
- **Prisma:** A modern ORM for Node.js and TypeScript. Compare it to traditional query builders or other ORMs like TypeORM. Key files: package.json, apps/mobile/package.json, prisma/schema.prisma
